## keyboard shortcut conventions (personal, cross-app)

[[Captured]] 2026-04-20. Applies wherever an app lets me rebind function keys.

- **F1–F5** — system-wide (OS-level actions, consistent across every app).
- **F6–F10** — app-specific.
  - **F8** — play / show / run / start (the "go" action for the app).
  - **F10** — full mode / zen mode / focus mode / distraction-free (whatever the app's equivalent is).
- **F11** — content handling, TBD (reserved).
- **F12** — content handling: screenshot.

Principle: when an app exposes a rebindable full-screen/zen toggle, it should be F10. When it exposes a rebindable run/play/execute, it should be F8.
